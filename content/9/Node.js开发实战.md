
# Node.js开发实战

> 导读：通过对express、jade、angular、mongodb等技术结合的一个示例代码，和大家分享涉及的技术点。

## express的使用

### 介绍
基于 Node.js 平台，快速、开放、极简的 web 开发框架。

### 安装

```
npm install express express-generator -g
```

### 生成项目脚手架代码

```
express myapp
```
